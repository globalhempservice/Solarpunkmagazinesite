## ğŸ¯ User Flows

### First Time Setup
```
1. User signs up
2. Click ME button
3. Click "My Profile"
4. See empty profile
5. Click "Edit Profile"
6. Upload avatar
7. Set display name
8. Write bio
9. Select roles
10. Select interests
11. Save
12. Profile complete!
```

### Profile View (Own)
```
1. Click ME button
2. Click "My Profile"
3. See full profile
4. See trust score (0 initially)
5. See stats cards
6. Can edit profile
```

### Profile View (Public)
```
1. Navigate to other user's profile
2. See their public info
3. No edit button
4. Can see trust score
5. Can see roles/interests
```
