## ðŸ’¡ PRO TIPS

### **Before Capturing:**
1. **Seed demo data:**
   - Create test organizations with logos
   - Add sample articles with images
   - Populate places on the globe
   - Generate achievements/transactions
2. **Clean up UI:**
   - Remove console errors
   - Hide dev tools
   - Full screen mode (F11)
3. **Set ideal state:**
   - Log in as user with progress (Level 15+, 5000 NADA, 10-day streak)
   - Open modals/panels to show features

### **During Capture:**
1. **Use browser extensions:**
   - Awesome Screenshot (Chrome)
   - Nimbus Screenshot (Firefox)
   - Full page screenshot capability
2. **Annotation (optional):**
   - Add arrows/highlights later in Figma/Photoshop
   - Label key features
3. **Consistency:**
   - Same zoom level (100%)
   - Same time of day (avoid real timestamps if possible)

### **After Capture:**
1. **Organize in folders:**
   ```
   /screenshots/
     /01_core/
     /02_apps/
     /03_admin/
     /04_mobile/
     /05_details/
   ```
2. **Resize for presentation:**
   - 1920x1080 â†’ 1440x810 (for slides)
   - Mobile: Keep aspect ratio
3. **Optimize file size:**
   - Use TinyPNG or ImageOptim
   - PNG for UI, JPG for photos
   - Target <500KB per image
